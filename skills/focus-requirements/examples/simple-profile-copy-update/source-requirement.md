# Reviewed Source Requirement

## Background

Users currently see "Account Settings" as the main title on the profile page.
Recent user interviews showed that people confuse this page with security and billing settings,
while the page actually only manages personal profile information.

## Requirement Summary

Update the profile page copy to make the page purpose clearer.

Changes requested:
- page title changes from `Account Settings` to `Profile Information`
- description text changes to `Manage your display name, avatar, and personal details`
- save success toast changes from `Settings updated` to `Profile updated`

## Scope Notes

- This is a web-only change
- No backend contract changes
- No analytics changes required
- No layout or interaction changes requested

## Success Expectations

- users understand the page is for profile editing
- support confusion about where to edit profile details should decrease

## Notes From Review

- Product copy is already approved
- This should land as a small incremental web change
