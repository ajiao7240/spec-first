# Cross-Layer 检查清单

## A. 跨层数据流（涉及 3+ 层时必做）

- [ ] 数据格式在层间转换正确
- [ ] 错误在层间正确传播
- [ ] 类型定义在各层一致
- [ ] 边界条件在各层处理一致

## B. 代码复用

### B1. 修改常量/配置时

- [ ] 检索所有使用该常量的位置
- [ ] 确认所有使用点都已更新
- [ ] 评估是否应提取为共享配置

### B2. 创建新工具函数时

- [ ] 搜索是否已存在类似函数
- [ ] 命名是否与现有风格一致
- [ ] 放置在正确模块位置

### B3. 批量修改后

- [ ] 确认影响范围完整
- [ ] 检查是否遗漏调用点
- [ ] 验证相关测试全部通过

## C. 导入/依赖路径

- [ ] 导入路径正确（相对/绝对）
- [ ] 无循环依赖
- [ ] 新文件位于正确目录

## D. 同层一致性

- [ ] 命名风格与同层组件一致
- [ ] API 签名与同层组件一致
- [ ] 错误处理与同层组件一致
