# 00-first 子 Agent 编排架构

> 说明 `spec-first:first` 标准模式下的并发波次、依赖关系与超时策略。

## 1. 波次总览

- 第一波：A1、A3、B、C1、C2 并行
- 第二波：A2（等待 A1），D 条件派发
- 第三波：A4（等待 A2 + B + D 完成）

## 2. 超时策略

| 策略 | 单 Agent 超时 | 整体并行阶段超时 |
|------|---------------|------------------|
| **标准模式** | 60s（单个 Agent） | 300s（整体） |

整体并行阶段最大超时：**300s**

## 3. 依赖链约束

- A2 依赖 A1 的模块结构与边界识别
- A4 等待 A2 + B + D 完成
- 若 D 未派发，A4 需要在输出中显式说明 DB 证据缺失

## 4. 降级策略

- 任一 Agent 超时后，不阻断其他 Agent 收口
- 主线程必须记录降级原因并在 README 中提示产物不完整
- 证据输出统一遵循 `quality-assurance-rules.md`
