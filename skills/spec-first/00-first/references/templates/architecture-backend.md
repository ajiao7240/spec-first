# 系统架构（后台服务）

> **模板类型**：backend
> **适用项目**：Spring Boot、Django、FastAPI、Gin、Laravel、Rails 等后端服务

---

## 分层架构

```
┌─────────────────────────────────────────────────────────────┐
│                        客户端层                              │
│                   (Web / Mobile / 第三方)                    │
└─────────────────────────────────────────────────────────────┘
                              ↓ HTTP/gRPC/WebSocket
┌─────────────────────────────────────────────────────────────┐
│                     API 网关 / 路由层                         │
│         (Nginx / API Gateway / 负载均衡)                     │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                    控制器层 (Controller)                     │
│              (请求处理、参数校验、响应封装)                    │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                     服务层 (Service)                         │
│                   (业务逻辑、事务控制)                         │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                   数据访问层 (Repository/DAO)                 │
│                    (数据持久化、缓存)                         │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                      数据存储层                              │
│           (MySQL / PostgreSQL / Redis / MongoDB)             │
└─────────────────────────────────────────────────────────────┘
```

## 中间件与基础设施

| 组件 | 用途 | 说明 |
|------|------|------|
| **消息队列** | RabbitMQ / Kafka / Redis Stream | 异步任务、事件驱动 |
| **缓存** | Redis / Memcached | 热数据缓存、分布式锁 |
| **搜索引擎** | Elasticsearch / Solr | 全文检索、日志分析 |
| **对象存储** | OSS / S3 / MinIO | 文件存储 |
| **监控** | Prometheus / Grafana | 指标采集、可视化 |

## 数据流向

```
用户请求 → API 网关 → 路由分发 → Controller
                               → 参数校验
                               → Service (业务逻辑)
                               → 事务控制
                               → Repository (数据访问)
                               → 数据库/缓存
                               ← 返回结果
                               ← 响应封装
← JSON / XML 响应
```

## 模块划分

| 模块 | 职责 |
|------|------|
| `controller/` 或 `api/` | API 端点定义 |
| `service/` 或 `domain/` | 核心业务逻辑 |
| `repository/` 或 `dao/` | 数据访问 |
| `model/` 或 `entity/` | 数据模型 |
| `dto/` 或 `vo/` | 数据传输对象 |
| `middleware/` | 中间件（认证、日志等） |
| `config/` | 配置管理 |
| `utils/` | 工具类 |

## 安全架构

| 机制 | 说明 |
|------|------|
| **认证** | JWT / OAuth2 / Session |
| **授权** | RBAC / ABAC 权限控制 |
| **防护** | 限流、熔断、降级、HTTPS |
