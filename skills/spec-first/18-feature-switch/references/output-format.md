```

目标: FEAT-20260305-001 | Skill 优化 | 💻 代码实现

是否继续？[Y/n]
```

## 错误消息

### Feature 不存在

```
❌ Feature 不存在: FEAT-20260399-999

可用的 Feature (最近 5 个):
1. FEAT-20260305-001 | Skill 优化
2. FEAT-20260304-002 | 文档补充
3. FEAT-20260303-003 | API 重构
4. FEAT-20260302-004 | 测试增强
5. FEAT-20260301-005 | Bug 修复

💡 提示: 使用 /spec-first:feature-list 查看所有 Feature
```

---

### 无匹配结果

```
❌ 未找到匹配的 Feature: "xyz"

搜索范围: 所有 Feature
匹配规则: 精确匹配 → 前缀匹配 → 模糊匹配

建议:
- 检查 Feature ID 拼写
- 使用完整的 Feature ID
- 运行 /spec-first:feature-list 查看可用 Feature
```

---

### 文件写入失败

```
❌ 切换失败: 无法写入 .spec-first/current

错误详情: EACCES: permission denied

建议操作:
1. 检查 .spec-first/ 目录权限
   chmod 755 .spec-first
2. 确保磁盘空间充足
   df -h
3. 检查文件是否被锁定
   lsof .spec-first/current
```
