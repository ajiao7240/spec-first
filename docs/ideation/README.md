## 当前主线

这组连载当前采用的总主线是：

`从 Vibe Coding 到工程系统：Spec-First 的完整演进叙事`

在方法论上，当前写法会把三层关系放在一起处理：

- `Vibe Coding`：强调先生成
- `Harness engineering`：强调先把 Agent 的环境、约束和验证回路搭起来
- `spec-first`：在 harness 思想之上，把跨宿主工作流、治理与知识复利产品化
