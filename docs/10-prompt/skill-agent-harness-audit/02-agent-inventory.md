## Inventory Judgment

- Code-review personas are the most mature agent family: many use JSON output, anchored confidence, and clear suppress rules.
- Document-review lens agents are useful but less stable: several return freeform prose, which increases synthesis load in `spec-doc-review`.
- Older expert profiles such as README writer, architecture strategist, performance oracle, security sentinel, pattern-recognition specialist, and data-integrity guardian are closer to personas than Harness components and should not be default reviewers without explicit trigger/input/output contracts.
- Agent final synthesis must remain parent-skill owned. Agents can emit findings, lens notes, or research packets; they should not decide workflow completion, merge readiness, or source mutation.
