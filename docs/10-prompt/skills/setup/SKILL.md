# 规格优先设置

规范优先工作流程的项目级配置。

## 当前状态

审核代理选择由 `spec:review` 工作流程自动处理，该工作流程使用基于差异内容的智能分层选择。代码审查不需要每个项目的配置。

如果调用此技能，请通知用户：

> 不再需要审阅代理配置 - `spec:review` 根据您的差异自动选择正确的审阅者。项目特定的审核上下文（例如，“我们服务 10k 请求/秒”或“监视 N+1 查询”）属于您项目的 CLAUDE.md 或 AGENTS.md，其中所有代理都已阅读它。

## 未来使用

该技能是为未来项目级配置需求而保留的，超出了审查代理选择的范围。
