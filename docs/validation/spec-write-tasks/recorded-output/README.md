# 记录输出证据

本目录保存 `spec-write-tasks` 实际输出或代表性输出的维护者侧证据。

每个 recorded output 需要包含：

- source plan path 与 source plan hash
- output path 与 output hash
- generation command 或 generation context
- adjudication status
- reviewer notes 与 residual missing evidence

这些文件是 source-maintainer evidence。它们不是 packaged skill runtime assets，也不应从 `skills/spec-write-tasks/SKILL.md` 引用。
