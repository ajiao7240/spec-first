## 项目总结

Stage Viewer 已完成核心功能开发，满足日常 Spec-First 开发流程的可视化需求。

**主要功能：**
- Feature 列表与搜索
- 阶段流转图与 Gate 状态
- 健康仪表盘（健康分 + 覆盖率 + 缺陷统计）
- 任务进度可视化
- 时间线视图（阶段耗时）
- 代码模块化 + 单元测试

**启动命令：** `spec-first viewer` 或 `node scripts/stage-viewer/server.js`

## API 端点

| 端点 | 说明 |
|------|------|
| `GET /api/features` | 获取所有 Feature 列表 |
| `GET /api/feature/:id` | 获取单个 Feature 详情 |
| `GET /api/feature/:id/tasks` | 获取任务进度数据 |
| `GET /api/feature/:id/metrics` | 获取覆盖率指标数据 |
| `GET /api/feature/:id/defects` | 获取缺陷统计数据 |
| `GET /api/feature/:id/gate-status` | 获取 Gate 状态 |
| `GET /api/feature/:id/timeline` | 获取时间线数据 |
