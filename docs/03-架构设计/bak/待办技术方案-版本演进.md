- 忽略其他平台

**v2 的错误修正**:
- ❌ 删除了 skill selector（核心能力）
- ❌ 删除了 skills-registry.json
- ❌ 删除了 hook 注入

**结果**: execution_profile 变成"死数据"

---

### v2 → v3: 恢复核心能力，但路径错误

**v3 的修正**:
- ✅ 恢复 skill selector
- ✅ 恢复 skills-registry.json
- ✅ 扩展到4个平台

**v3 的致命缺陷**:
```json
// ❌ 错误设计
{
  "selected_skills": [
    ".spec-first/skills/tdd.md"  // 路径不存在
  ]
}
```

**问题**:
1. ❌ `.spec-first/skills/` 路径根本不存在
2. ❌ 文件路径是平台相关的
3. ❌ 与实际代码完全脱节

---

### v3 → v4: 修正路径体系

**v4 的关键修正**:
```json
// ✅ 正确设计
{
  "selected_skills": [
    "test-driven-development"  // skill ID
  ]
}
```

**正确的架构**:
- ✅ task.json 只存储 skill IDs（平台无关）
- ✅ 各平台负责 skill ID → 文件路径的映射
- ✅ 每个平台可以有不同的目录结构

---

## 架构对比

### v3 架构（错误）

```
execution_profile
  ↓
skill selector
  ↓
selected_skills = [".spec-first/skills/tdd.md"]  // ❌ 路径
  ↓
各平台直接读取路径注入  // ❌ 路径不存在
```

**问题**:
- 路径是平台相关的
- `.spec-first/skills/` 不存在
- 与实际代码脱节

### v4 架构（正确）

```
execution_profile
  ↓
skill selector
  ↓
selected_skills = ["test-driven-development"]  // ✅ ID
  ↓
各平台映射到实际路径
  - Claude: ".claude/skills/tdd/SKILL.md"
  - Codex: ".codex/skills/tdd/SKILL.md"
  - iFlow: ".iflow/skills/tdd/SKILL.md"
  ↓
注入到 prompt
```

**优势**:
- skill ID 是平台无关的
- 各平台灵活决定文件位置
- 与实际代码一致

## 关键差异

### skills-registry.json

**v3**:
```json
{
  "skills": [
    {
      "id": "test-driven-development",
      "path": ".spec-first/skills/tdd.md",  // ❌ 错误路径
      ...
    }
  ]
}
```

**v4**:
```json
{
  "skills": [
    {
      "id": "test-driven-development",
      // ✅ 无 path 字段
      ...
    }
  ]
}
```

### skill selector 输出

**v3**:
```python
def select_skills(...):
    return [skill['path']]  # ❌ 返回路径
    # [".spec-first/skills/tdd.md"]
```

**v4**:
```python
def select_skills(...):
    return [skill['id']]  # ✅ 返回 ID
    # ["test-driven-development"]
```

### 平台注入

**v3**:
```python
selected_skills = task_data.get('selected_skills', [])
# [".spec-first/skills/tdd.md"]
for path in selected_skills:
    content = Path(path).read_text()  # ❌ 路径不存在
```

**v4**:
```python
selected_skill_ids = task_data.get('selected_skills', [])
# ["test-driven-development"]

SKILL_PATH_MAP = {
    "test-driven-development": ".claude/skills/tdd/SKILL.md"
}

for skill_id in selected_skill_ids:
    path = SKILL_PATH_MAP.get(skill_id)  # ✅ 映射到实际路径
    content = Path(path).read_text()
```
