## 修正方案

### 修正点 #1: skill selector 输出 Skill IDs

```python
# skill_selector.py
def select_skills(execution_profile: dict, registry_path: str) -> list[str]:
    """根据 execution_profile 选择 skill bundle

    返回: skill ID 列表（不是文件路径）
    """
    registry = _load_registry(registry_path)
    task_mode = execution_profile.get('task_mode', 'default')
    surface = execution_profile.get('surface')

    selected_ids = []
    for skill in registry['skills']:
        if _is_applicable(skill, task_mode, surface):
            selected_ids.append(skill['id'])  # ✅ 返回 ID，不是 path

    return selected_ids
```

### 修正点 #2: 删除 skills-registry.json 中的 path 字段

```json

      // ✅ 删除 "path" 字段

  ]

```

### 修正点 #3: 各平台实现路径映射

1. 定义 skill ID → 文件路径的映射
2. 在注入时读取 skill ID 并映射到实际路径
