## 一、架构原则：DDD 分层

```
┌─────────────────────────────────────────────────────────┐
│ Domain Layer (领域层) - 只定义规则                        │
│ - id-rules.ts: ID 类型、正则、链路规则                    │
│ - coverage-rules.ts: 覆盖率计算规则                       │
│ - gate-rules.ts: 阶段推进规则                            │
└─────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────┐
│ Infrastructure Layer (基础设施层) - 只做技术实现          │
│ - parseMatrix: 解析 Markdown + 格式校验                  │
│ - validateId: ID 格式校验服务                            │
│ - validateChain: 链路合法性校验服务                       │
└─────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────┐
│ Application Layer (应用层) - 只做业务逻辑                │
│ - checkMatrix: 链路完整性应用服务                         │
│ - getCoverage: 覆盖率计算应用服务                         │
│ - validatePrd: PRD 质量评估服务                          │
└─────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────┐
│ Presentation Layer (表现层) - 只做编排决策               │
│ - evaluateGate: 质量门禁编排服务                         │
│   聚合 checkMatrix + getCoverage + validatePrd          │
│   应用阶段推进规则，返回通过/失败决策                      │
└─────────────────────────────────────────────────────────┘
```

```typescript
function parseMatrixContent(content: string): MatrixRow[] {
  const rows: MatrixRow[] = [];

  for (const cells of parseMarkdownTable(content)) {
    const id = cells[0];
    const upstream = parseRefList(cells[4]);
    const downstream = parseRefList(cells[5]);

    // 聚焦：格式校验
    const validation = validateId(id);
    if (!validation.valid) {
      throw new Error(`Invalid ID: ${id}`);
    }

    // 聚焦：引用合法性校验
    for (const ref of [...upstream, ...downstream]) {
      if (!validateId(ref).valid) {
        throw new Error(`Invalid reference: ${ref} in ${id}`);
      }
    }

    rows.push({
      id,
      type: validation.type!,
      title: cells[2],
      status: normalizeStatus(cells[3]),
      upstream,
      downstream,
    });
  }

  return rows;
}
```

**聚焦点**：
- ✅ 解析 Markdown 表格

**文件**：`src/core/trace-engine/id-validator.ts`

```typescript
export function validateId(id: string): IdValidationResult {
  for (const [type, rule] of Object.entries(ID_RULES)) {
    if (rule.regex.test(id)) {
      return { valid: true, type: type as IdType };
    }
  }
  return { valid: false, error: `Unknown ID: "${id}"` };
}

export function validateChain(id: string, upstream: string[]): string[] {
  const validation = validateId(id);
  if (!validation.valid) return ['Invalid ID format'];

  const rule = ID_RULES[validation.type!];
  const errors: string[] = [];

  for (const upId of upstream) {
    const upValidation = validateId(upId);
    if (!upValidation.valid) {
      errors.push(`Invalid upstream: ${upId}`);
      continue;
    }
    if (!rule.upstream.includes(upValidation.type!)) {
      errors.push(`${validation.type} cannot have ${upValidation.type} as upstream`);
    }
  }

  return errors;
}
```

**聚焦点**：
- ✅ 校验 ID 格式
- ✅ 校验链路合法性
- ❌ 不做完整性判断

```typescript
export function checkMatrix(
  featureId: string,
  projectRoot: string
): MatrixCheckResult {
  const rows = parseMatrix(featureId, projectRoot);
  const errors: string[] = [];

  // 聚焦：链路完整性
  for (const row of rows) {
    const chainErrors = validateChain(row.id, row.upstream ?? []);
    errors.push(...chainErrors.map(e => `${row.id}: ${e}`));
  }

  // 聚焦：孤儿检测
  const orphans = rows.filter(r =>
    r.type !== 'Feature' && r.type !== 'REQ' &&
    (!r.upstream || r.upstream.length === 0)
  );

  return {
    total: rows.length,
    errors,
    orphans,
    valid: errors.length === 0 && orphans.length === 0,
  };
}
```

**聚焦点**：
- ✅ 链路完整性检查
- ✅ 孤儿检测
- ❌ 不做质量评估
- ❌ 不做阶段判断

### Layer 4: Presentation (表现层)

**职责**：编排决策，不做具体计算

**文件**：`src/core/gate-engine/gate-evaluator.ts`

```typescript
export function evaluateGate(
  featureId: string,
  stage: Stage,
  projectRoot: string
): GateResult {
  // 聚焦：聚合应用层服务
  const rows = parseMatrix(featureId, projectRoot);
  const coverage = getCoverage(featureId, projectRoot);
  const matrixCheck = checkMatrix(featureId, projectRoot);

  const ctx: GateContext = {
    featureId,
    stage,
    projectRoot,
    rows,
    coverage,
  };

  // 聚焦：应用阶段规则
  const conditions = GATE_CONDITIONS[stage] || [];
  const results: GateConditionResult[] = [];

  // 前置：链路完整性
  if (!matrixCheck.valid) {
    results.push({
      id: 'G-MATRIX',
      description: 'Matrix integrity',
      pass: false,
      detail: matrixCheck.errors.join('; '),
    });
  }

  // 聚焦：评估质量指标
  for (const cond of conditions) {
    results.push({
      id: cond.id,
      description: cond.description,
      ...cond.evaluate(ctx),
    });
  }

  // 聚焦：返回决策
  return {
    pass: results.every(r => r.pass),
    results,
  };
}
```

**聚焦点**：
- ✅ 聚合应用层服务
- ✅ 应用阶段推进规则
- ✅ 返回通过/失败决策
- ❌ 不做具体计算
- ❌ 不做格式校验
