# skill治理审查实施计划

> **给 Claude：** 必须使用 `superpowers:executing-plans`，按任务逐步执行本计划。

**目标：** 让 `spec-first` 的端到端 skill 流程在运行时可强制执行，并且在完整 skill 目录范围内可审计。

**架构方案：** 先收紧 runtime hard-gate，确保所有绑定阶段的 skill 在错误阶段都会被阻断；再修复当前影响可发现性和引用完整性的少量 skill 资产问题；最后补齐 catalog 级测试，防止规则再次静默漂移。

**技术栈：** TypeScript、Vitest、Markdown skill 资产
