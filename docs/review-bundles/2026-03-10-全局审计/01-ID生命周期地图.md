## 七、关键发现

### 7.1 ID 生成无锁保护

**证据**:
- Feature ID 生成有锁 (withRegistryLock)
- 其他 ID 生成无锁 (nextId 直接写文件)

**风险**: 并发调用 nextId 可能产生重复 ID

---

### 7.2 ID 传播无校验

**证据**: readCurrentFeature() 直接读取文件内容，不校验格式

**风险**: 文件被手动修改为无效 ID 时，运行时错误

---

### 7.3 矩阵解析静默降级

**证据**: matrix.ts:160 使用 `validation.type ?? 'Feature'`

**风险**: 无效 ID 被错误识别为 Feature 类型

---

### 7.4 循环依赖静默处理

**证据**: upstream-lineage.ts:23 检测到循环返回空集合

**风险**: 用户无法发现循环依赖
