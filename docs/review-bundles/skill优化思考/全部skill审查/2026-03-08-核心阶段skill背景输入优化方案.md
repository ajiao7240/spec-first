### 2.2 短期优化（质量提升）

#### 优化 1: 统一降级策略格式

**所有 4 个 skill 补充统一的降级策略章节**:

```markdown
### 背景状态定义

| 状态 | 条件 | 行为 |
|------|------|------|
| **full** | `.spec-first/runtime/first/` 完整且健康 | 使用 {stage}-view 完整执行 |
```

```markdown
## 执行流程

2. 检查 `stage-views.json` 是否包含 {stage}-view
3. 确定 background_input_status（full/degraded/blind）
4. 输出背景状态：
   ```
   📊 背景输入状态: {status}
   - Runtime 真源: {存在/缺失}
   - Stage View: {可用/不可用}
   - 降级影响: {无/中等/高}
   ```
5. 若为 degraded/blind，询问用户是否继续
```
