# 2026-03-07 17-feature-list skill 审查与优化建议

## 1. skill 要解决什么问题

`17-feature-list` 解决的是：

1. 列出当前项目中的全部 Feature
2. 帮用户找到可切换或可继续处理的目标

一句话：

> `17-feature-list` 解决“当前项目里有哪些 Feature”。 

## 2. 当前实现是否合理

整体判断：功能单一、合理、没有明显问题。

## 3. 当前主要问题

最大问题不是功能本身，而是命令面：

- 它和 `feature-current`、`feature-switch` 天然属于同一组操作
- 独立成一级命令会增加用户记忆成本

## 4. 推荐优化方向

建议并入：

- `feature list`

## 5. 当前审查结论

- 职责定位：正确
- 当前合理性：高
- 主要问题：独立入口价值偏低
- 最佳优化方向：并入 `feature` 命令族

一句话结论：

> `17-feature-list` 没问题，但更适合做 `feature` 家族里的一个子动作。
