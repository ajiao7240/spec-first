### ✅ 问题 2: gate check 将 warning 写入 Failed Conditions (Medium)

**状态**: ✅ **确认存在**

**位置**: `src/cli/commands/gate.ts:212-224`

**问题代码**:
```typescript
function appendFixStepsToFindings(...) {
  const failedConditions = conditions
    .filter((condition) => condition.status === 'FAIL')  // ❌ 包含 warning
    .map(...);
}
```

**问题**:
- gate.ts:220 过滤 `status === 'FAIL'` 包含了 `blocking: false` 的 warning
- 导致 Gate 通过但 findings.md 记录为失败条件

**正确逻辑**:
```typescript
.filter((condition) => condition.status === 'FAIL' && condition.blocking !== false)
```
