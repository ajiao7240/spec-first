# 本地源码安装指南

## 前置要求

- Node.js >= 16
- npm >= 8
- Claude Code CLI 已安装

## 安装步骤

### 1. 克隆仓库

```bash
git clone https://github.com/sunrain520/spec-first.git
cd spec-first
```

### 2. 安装依赖（可选）

```bash
npm install
```

### 3. 链接到全局

```bash
npm link
```

这会将 spec-first 链接到 Claude Code 的插件目录。

### 4. 验证安装

```bash
# 检查插件目录
ls ~/.claude/plugins/

# 在 Claude Code 中测试
/spec:brainstorm --help
```

如果看到帮助信息，说明安装成功。

## 测试安装

运行 E2E 测试：

```bash
npm test
```

应该看到：
```
=== E2E 测试全部通过 ✓ ===
```

## 开发模式

如果需要修改代码：

### 1. 修改文件

编辑 `skills/` 或 `scripts/` 中的文件

### 2. 无需重新安装

由于使用了 `npm link`，修改会立即生效

### 3. 测试修改

```bash
# 运行测试
npm test

# 在 Claude Code 中测试
/spec:brainstorm "测试功能"
```

## 卸载

```bash
# 取消链接
npm unlink -g @spec-first/framework

# 删除源码
cd ..
rm -rf spec-first
```

## 故障排查

### 问题1：npm link 失败

```bash
# 使用 sudo
sudo npm link
```

### 问题2：找不到 skills

```bash
# 检查链接
ls -la ~/.claude/plugins/

# 重新链接
npm unlink -g @spec-first/framework
npm link
```

### 问题3：修改不生效

```bash
# 重启 Claude Code
```
