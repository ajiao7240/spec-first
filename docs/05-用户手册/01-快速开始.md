# 快速开始指南

## 安装

### 方式1：从 npm 安装

```bash
npm install -g @spec-first/framework
```

### 方式2：从源码安装

```bash
git clone https://github.com/sunrain520/spec-first.git
cd spec-first
npm link
```

## 验证安装

```bash
# 在 Claude Code 中验证
/spec:brainstorm --help
```

## 第一个任务

### 1. 启动 Brainstorm

```bash
/spec:brainstorm "添加用户登录功能"
```

Claude 会引导你：
- 探索需求
- 讨论方案
- 确定范围

### 2. 制定计划

```bash
/spec:plan
```

生成实施计划到 `.claude/tasks/<task-id>/02-plan.md`

### 3. 执行实施

```bash
/spec:work
```

按计划执行开发工作

### 4. 质量评审

```bash
/spec:review
```

自动评审代码质量

### 5. 知识沉淀

```bash
/spec:compound
```

提取可复用知识
