# Spec-First 用户手册

欢迎使用 Spec-First Framework！

当前手册对应 `spec-first` 的 npm CLI 模型：

- `npm install -g spec-first`
- `spec-first doctor`
- `spec-first init --claude`
- `/spec:*` 命令来自项目内 `.claude/commands/spec/`
