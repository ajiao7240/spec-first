# 其他平台配置

spec-first 支持多种 AI 编码工具，以下是简要配置说明。

## iFlow CLI

### 平台简介
iFlow CLI 是功能完整的 AI 编码工具，支持 Python 钩子和多代理流水线。

### 配置目录
`.iflow/`

### 文件结构
```
.iflow/
├── agents/           # 多代理配置
├── commands/spec/    # Slash 命令
├── hooks/            # Python 钩子
└── settings.json     # 配置文件
```

### 初始化
```bash
npx spec-first init --iflow
```

### 特点
- 完整功能支持，与 Claude Code 类似
- 支持 Python 钩子
- 支持多代理流水线

---

## Codex

### 平台简介
OpenAI Codex 的 CLI 工具，使用 Skills 格式。

### 配置目录
- `.codex/` - Codex 特定配置
- `.agents/skills/` - 跨平台共享 Skills（开放标准）

### 文件结构
```
.agents/skills/       # 共享 Skills
├── before-dev/SKILL.md
├── brainstorm/SKILL.md
├── check/SKILL.md
└── ...

.codex/
├── agents/           # TOML 格式代理
├── skills/           # Codex 特定 Skills
├── hooks/            # 钩子脚本
├── hooks.json        # 钩子配置
└── config.toml       # Codex 配置
```

### 初始化
```bash
npx spec-first init --codex
```

### 特点
- 支持 `.agents/skills/` 开放标准
- TOML 格式配置
- Python 会话钩子

## 多平台初始化

可以同时初始化多个平台：

```bash
# Claude Code + Cursor
npx spec-first init --claude --cursor

# 所有主流平台
npx spec-first init --claude --cursor --opencode --codex

# 交互式选择
npx spec-first init
```
