## 安装

```bash
# 全局安装（推荐）
npm install -g spec-first@latest

# 查看版本
spec-first --version

# 查看帮助
spec-first --help
```

---

## 初始化

### 基础命令

```bash
# 最小化初始化
spec-first init -u <用户名>

# 指定平台
spec-first init --cursor --claude --codex -u <用户名>

# 快速初始化（跳过提示）
spec-first init -y -u <用户名>

# 强制覆盖
spec-first init -f -u <用户名>
```

### 平台选项

| 选项 | 平台 |
|------|------|
| `--cursor` | Cursor |
| `--claude` | Claude Code |
| `--opencode` | OpenCode |
| `--iflow` | iFlow |
| `--codex` | Codex |
| `--kilo` | Kilo |
| `--kiro` | Kiro |
| `--gemini` | Gemini CLI |
| `--antigravity` | Antigravity |
| `--qoder` | Qoder |
| `--codebuddy` | CodeBuddy |

### 其他选项

| 选项 | 说明 |
|------|------|
| `-y, --yes` | 跳过提示，使用默认配置 |
| `-f, --force` | 强制覆盖现有文件 |
| `-s, --skip-existing` | 跳过已存在的文件 |
| `-t, --template <name>` | 使用远程模板 |
| `--monorepo` | 强制启用 monorepo 模式 |

## 更新

### 更新 CLI

```bash
# npm
npm install -g spec-first@latest

# pnpm
pnpm update -g spec-first

# yarn
yarn global upgrade spec-first
```

### 更新项目配置

```bash
# 预览变更
spec-first update --dry-run

# 交互式更新
spec-first update

# 强制更新
spec-first update --force

# 跳过所有
spec-first update --skip-all

# 创建 .new 副本
spec-first update --create-new

# 应用迁移
spec-first update --migrate
```

## 卸载

```bash
# npm 卸载
npm uninstall -g spec-first

# pnpm 卸载
pnpm uninstall -g spec-first

# yarn 卸载
yarn global remove spec-first

# 删除项目配置
rm -rf .spec-first/
```

## 常见场景

### 场景 1: 新项目初始化

```bash
cd /path/to/new-project
spec-first init --cursor --claude -u myname
```

### 场景 2: 现有项目集成

```bash
cd /path/to/existing-project
spec-first init -s -u myname  # 跳过已存在的文件
```

### 场景 3: 添加新平台支持

```bash
spec-first init --codex -u myname -f
```

### 场景 4: 使用模板

```bash
spec-first init -u myname \
  -t electron-fullstack \
  --registry gh:myorg/spec-templates
```

### 场景 5: 更新到最新版本

```bash
# 1. 更新 CLI
npm install -g spec-first@latest

# 2. 更新项目
cd /path/to/project
spec-first update
```

## 故障排除

### 命令找不到

```bash
# 检查 PATH
npm config get prefix
export PATH="$(npm config get prefix)/bin:$PATH"
```

### 权限错误

```bash
# 修改目录权限
chown -R $USER:$USER /path/to/project
```

### 版本冲突

```bash
# 检查版本
spec-first --version
cat .spec-first/.version

# 更新到最新
npm install -g spec-first@latest
spec-first update
```

### 清理缓存

```bash
npm cache clean --force
npm uninstall -g spec-first
npm install -g spec-first@latest
```
