# 五、业务域依赖

【涉及datatype/交易时间/接口变更/风控调整的必填】

## 5\.1 依赖项总览

|依赖方|改动点|接口动作|生效时效|影响范围|对接人|
|---|---|---|---|---|---|
|行情侧||新增接口/现有接口需调整/复用现有/无|实时/重启/T\+1/缓存N分钟/手动触发|||
|柜台侧||||||
|中台侧||||||
|前端侧（APP/PC）||||||
|风控侧||||||
|第三方||||||

**接口动作枚举：**

- **新增接口**：依赖方需要从0开发新接口

- **现有接口需调整**：依赖方现有接口的传参/回参/逻辑要改

- **复用现有**：直接调用现有接口，依赖方无任何改造

- **无**：不涉及接口调用

**生效时效枚举：**

- **实时生效**：配置/接口推送后立即生效

- **重启生效**：依赖方服务重启后生效（必须标注预计重启窗口）

- **T\+1**：次日批跑后生效

- **缓存N分钟**：N分钟后自动刷新（必须标注N值）

- **手动触发**：需人工执行（必须标注执行人和动作）
