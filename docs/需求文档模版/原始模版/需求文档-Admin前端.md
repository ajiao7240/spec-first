# 五、通用规则（全局适用）

## 5\.1 布局规范

- 顶部：Logo \+ 面包屑 \+ 当前用户 \+ 退出

- 左侧：折叠菜单（默认展开一级）

- 右侧：主内容区（列表 / 详情 / 表单）

## 5\.2 列表页通用规则

|**项**|**规则**|
|---|---|
|分页|默认 20 条/页，可选 10/20/50/100|
|排序|默认按\&\#34;创建时间\&\#34;倒序|
|筛选|多条件组合，\&\#34;重置\&\#34;按钮清空所有筛选|
|搜索|支持回车触发，输入框最大长度 50|
|空状态|显示插画 \+ \&\#34;暂无数据\&\#34;|
|加载态|表格 Skeleton，按钮 Loading|
|错误态|\&\#34;加载失败，点击重试\&\#34;|

## 5\.3 表单通用规则

|**项**|**规则**|
|---|---|
|必填校验|失焦校验 \+ 提交校验|
|错误提示|字段下方红色文案|
|提交反馈|成功 Toast / 失败 Toast，含明确文案|
|防重复提交|提交后按钮 Loading 禁用|
|离开拦截|表单有改动未保存时，弹窗二次确认|

## 5\.4 异常状态（必覆盖）

- **空状态**：无数据时的展示

- **加载态**：骨架屏 / Loading

- **错误态**：网络异常、接口失败

- **无权限**：403 页面

- **404**：路由不存在

## 5\.5 文案与格式规范

|**类型**|**规则**|
|---|---|
|时间|YYYY\-MM\-DD HH:mm:ss|
|金额|¥ 1,234\.56（保留两位小数）|
|手机号|138\*\*\*\*8888（脱敏）|
|长文本|超出 20 字省略号，悬浮显示全文|
