### 2.6 CLI 状态面板（`status.ts`）：TS2322 修复

**背景**：`decideNextStep` 的 `todoState.items` 构造时缺少 `title` 字段，导致 TypeScript 编译错误。

**修复内容**：
```typescript
// 修复前（缺少 title）
items: taskPlan.tasks.map((task) => ({
  id: task.id,
  status: task.status === 'complete' ? 'done' : ...
}))

// 修复后（补充 title）
items: taskPlan.tasks.map((task) => ({
  id: task.id,
  title: task.title,  // ← 补充
  status: task.status === 'complete' ? 'done' : ...
}))
```

**验证**：`npm run typecheck` 无错误输出，所有测试通过。

**评级**: ✅ 类型错误已修复，实现闭环完整
