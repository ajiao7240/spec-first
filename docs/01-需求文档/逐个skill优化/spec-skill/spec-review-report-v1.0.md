### C10：复杂度分类时序悖论 ✅ 已修复

- 位置：§15 PRD 模板 YAML `complexity` 字段 vs §4.0 Phase 0.2 vs §4.2 Step 2
- R2 矛盾：Phase 0 时 complexity 必然为空，但 PRD 模板要求填写
- R3 修复：

### C11：REQ-PRD-xxx → FR 映射关系缺失 ✅ 已修复

- 位置：§5 FR 定义中无上游引用
- R2 断裂链路：`用户需求 → REQ-PRD-xxx → ??? → FR-SPEC-OPT-xxx → AC`

  2. FR-011 增加 `upstream: [REQ-PRD-*]` 字段
