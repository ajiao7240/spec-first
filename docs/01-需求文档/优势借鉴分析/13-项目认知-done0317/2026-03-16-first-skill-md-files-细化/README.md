# First Skill MD 文件逐项细化目录

本目录按两组细化 `first` 相关 md 文件的重构方案：

1. `skill-docs/`
   - 对应 `skills/spec-first/00-first/` 下的 `SKILL.md` 与 `references/*.md`
2. `docs-first/`
   - 对应 `docs/first/` 已有文档 + 目标新增专题文档

每个文件的细化说明统一包含：

- 当前角色
- 当前问题
- 改造动作
- 新定位
- Runtime 来源
- 章节建议
- 验收标准
