## 一、总目标

当前 `first` 的改造目标已经明确为：

1. 从旧的文档生成体系，重构为 **runtime-first 项目认知编译器**
2. 采用 **单一标准模式**，不再区分 quick / deep
3. 建立三层文档体系：
   - 基础 canonical docs
   - 正式专题文档
   - 条件型正式文档
4. 让 `06_wrap_up -> 07_release -> 08_done` 能自动更新新的完整文档体系
