## дёғгҖҒз»“и®ә

еҪ“еүҚ First Skill зҡ„ Runtime вҶ’ Docs жҠ•еҪұдҪ“зі»е·Іеҹәжң¬е»әз«ӢпјҢдҪҶеӯҳеңЁд»ҘдёӢж ёеҝғй—®йўҳпјҡ

1. **еӯӨе„ҝж–Үд»¶**пјҡ`modules.json` жңӘиў«дҪ“зі»иҝҪиёӘ
2. **ж–ҮжЎЈи„ұиҠӮ**пјҡSKILL.md дёҺе®һйҷ…дә§зү©дёҘйҮҚдёҚз¬Ұ
3. **жһ¶жһ„дёҚдёҖиҮҙ**пјҡ4 дёӘ Legacy дә§зү©иө°зӢ¬з«Ӣз”ҹжҲҗи·Ҝеҫ„
4. **жүҝиҜәжңӘе…‘зҺ°**пјҡ6 дёӘ Deep жЁЎејҸдә§зү©жңӘе®һзҺ°

е»әи®®жҢүдјҳе…Ҳзә§йҖҗжӯҘдҝ®еӨҚпјҢе…Ҳи§ЈеҶі P1/P2 зҡ„ж–ҮжЎЈдёҖиҮҙжҖ§й—®йўҳпјҢеҶҚжҺЁиҝӣ Legacy дә§зү©и§„иҢғеҢ–гҖӮ
