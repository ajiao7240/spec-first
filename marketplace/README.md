# spec-first Marketplace

Downloadable templates and specs for spec-first users.

## Structure

```
marketplace/
├── skills/      # Claude Skills (.claude/skills/)
├── commands/    # Slash Commands (.claude/commands/)
├── agents/      # Sub-agent definitions (.claude/agents/)
└── specs/       # Spec templates (.spec-first/spec/)
```

## Usage

### Install via npx skills

```bash
npx skills add sunrain520/spec-first/marketplace
```

### Manual Installation

Copy the desired template to your project's corresponding directory.
