# Release Note — v0.5.77

## Feature: Stage Viewer 页面优化

### 新增功能
- Gate 条件支持 [WARN]/[FAIL]/[OK] 状态标识
- 覆盖率指标精简为 C3/C4/C6/C8/C9 五个核心指标
- 健康分基于 5 指标重新计算，优化颜色阈值
- 健康分卡片内显示当前 profile

### 变更说明
- 已下线指标（C1/C2/C5/C7）不再显示
- 健康分颜色阈值调整：≥90 绿色，70-89 黄色，<70 红色

### 影响范围
- stage-viewer 前端页面

### 升级说明
无需特殊操作，刷新页面即可看到新功能。
