## 概述

优化 Stage Viewer 本地工具页面的加载和渲染性能，提升用户体验。

## 主要改进

### 性能优化

- **首屏加载优化**: FCP ≤ 1s, TTI ≤ 1.5s
- **CSS 优化**: 关键 CSS 内联，非关键 CSS 延迟加载
- **JavaScript 优化**: 搜索防抖、DOM 增量更新
- **缓存机制**: 内存缓存 TTL 30s，减少重复请求
- **列表渲染**: CSS contain 优化，100+ 条目渲染 < 300ms

### 用户体验

- **骨架屏**: 加载时显示友好的占位符
- **强制刷新**: 提供手动刷新按钮绕过缓存
- **交互响应**: 搜索响应 < 100ms

## 变更文件

- `scripts/stage-viewer/index.html` - 内联关键 CSS、骨架屏
- `scripts/stage-viewer/styles.css` - 样式优化
- `scripts/stage-viewer/app.js` - 缓存、防抖、增量更新

## 测试覆盖

- 单元测试: 18 个测试用例全部通过
- 冒烟测试: 核心功能验证通过
- 覆盖率: C1-C9 全部 100%
