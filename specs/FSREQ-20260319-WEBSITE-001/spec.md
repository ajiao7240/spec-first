## 功能需求 (FR)

### FR-WEBSITE-001: 首页展示

**描述**：展示产品简介、核心价值主张、快速开始入口

**验收条件 (AC)**：
- [ ] AC-001-01: Hero 区域包含产品名称、一句话描述、核心价值主张
- [ ] AC-001-02: 提供"快速开始"和"查看文档"两个 CTA 按钮
- [ ] AC-001-03: Hero 区域下方展示 3-4 个核心特性卡片
- [ ] AC-001-04: 页面首屏加载时间 < 2 秒

### FR-WEBSITE-002: 交互式终端演示

**描述**：在首页嵌入交互式终端，展示 spec-first 命令使用

**验收条件 (AC)**：
- [ ] AC-002-01: 终端组件支持展示预设命令序列（如 `spec-first init`、`spec-first status`）
- [ ] AC-002-02: 用户可点击运行演示命令，查看模拟输出
- [ ] AC-002-03: 终端样式与代码高亮正常显示
- [ ] AC-002-04: 移动端终端组件自适应或降级为 GIF 动图

## 非功能需求 (NFR)

### NFR-WEBSITE-001: 性能

- 页面首屏加载时间 < 2 秒
- 静态资源压缩（HTML/CSS/JS minification）
- 图片使用 WebP 格式或懒加载

### NFR-WEBSITE-002: SEO

- 所有页面包含正确的 meta 标签（title、description、og:*）
- 使用语义化 HTML 标签
- 生成 sitemap.xml

### NFR-WEBSITE-003: 无障碍

- 遵循 WCAG 2.1 AA 标准
- 所有图片包含 alt 属性
- 键盘导航支持

### NFR-WEBSITE-004: 安全

- HTTPS 部署
- 外部链接使用 `rel="noopener noreferrer"`
