# Spec — FSREQ-20260309-HOMEPAGE-001

> Spec-First Viewer 首页样式优化

## 功能需求 (FR)

### FR-VIS-001: 视觉层次优化

**描述**: 改善 Viewer 界面的信息层级结构和色彩系统

**验收标准**:
- AC-VIS-001-01: 标题、正文、辅助文本的字号和颜色对比度符合 WCAG AA 标准
- AC-VIS-001-02: 间距规范统一，section 间距为 20px，卡片内边距为 12-16px
- AC-VIS-001-03: 关键信息（健康分、阶段状态）视觉权重明显高于辅助信息
- AC-VIS-001-04: 状态色（成功/警告/危险）辨识度清晰，色差 ≥ 3:1

**优先级**: P0

---

### FR-INT-001: 交互体验提升

**描述**: 增强悬停反馈和焦点状态，提升操作流畅度

**验收标准**:
- AC-INT-001-01: Feature 列表项悬停时有明显视觉反馈（背景色变化 + 位移动画）
- AC-INT-001-02: 所有可点击元素悬停时显示 pointer 光标
- AC-INT-001-03: 过渡动画时长为 150-200ms，使用 ease 缓动函数
- AC-INT-001-04: 键盘焦点指示清晰可见，outline 宽度 ≥ 2px

**优先级**: P0

---

### FR-VIZ-001: 数据可视化增强

**描述**: 优化健康分环形图、覆盖率柱状图、缺陷统计卡片的视觉呈现

**验收标准**:
- AC-VIZ-001-01: 健康分环形图使用渐变色，分数数值字号 ≥ 24px
- AC-VIZ-001-02: 覆盖率柱状图悬停时显示具体数值
- AC-VIZ-001-03: 缺陷统计卡片有阴影效果，严重等级色块区分明显
- AC-VIZ-001-04: 统计数字使用等宽字体，字号 ≥ 18px

**优先级**: P0

---

### FR-LAY-001: 整体布局优化

**描述**: 优化侧边栏和主区域的布局与间距

**验收标准**:
- AC-LAY-001-01: 侧边栏搜索框有明显焦点状态
- AC-LAY-001-02: Feature 列表项间距统一为 8px
- AC-LAY-001-03: 主区域 section 间距统一为 20px
- AC-LAY-001-04: 滚动条样式统一，宽度为 6px

**优先级**: P1

---

## 非功能需求 (NFR)

### NFR-PERF-001: 性能要求

**描述**: CSS 文件大小和渲染性能要求

**验收标准**:
- AC-PERF-001-01: styles.css 文件大小 ≤ 30KB
- AC-PERF-001-02: 避免使用复杂的 CSS 选择器（嵌套层级 ≤ 3）

**优先级**: P1

---

### NFR-COMPAT-001: 兼容性要求

**描述**: 浏览器兼容性要求

**验收标准**:
- AC-COMPAT-001-01: Chrome/Edge 最新版本显示正常
- AC-COMPAT-001-02: Firefox 最新版本显示正常
- AC-COMPAT-001-03: Safari 最新版本显示正常
- AC-COMPAT-001-04: 桌面端宽度 ≥ 1280px 时显示正常

**优先级**: P1

---
