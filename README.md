# Spec-First Framework

AI-assisted work framework for Claude Code with five-step loop.

## Installation

```bash
npm install -g @spec-first/framework
```

## Quick Start

```bash
# 1. Brainstorm
/spec:brainstorm "新功能想法"

# 2. Plan
/spec:plan

# 3. Work
/spec:work

# 4. Review
/spec:review

# 5. Compound
/spec:compound
```

## Core Skills

- `spec:brainstorm` - 需求探索和方案设计
- `spec:plan` - 实施计划制定
- `spec:work` - 执行落地
- `spec:review` - 质量评审
- `spec:compound` - 知识沉淀

## Testing
